# SeaBattle

Интерактивная игра "морской бой"

Реализовано:
* GUI
* Базовые механики
* Кастомные размеры поля
* Кастомные корабли и их количество
* Мины и минные тральщики
* Два режима игры
  * С другим игркоком (hot seat)
  * С ботом (на выбор 4 уровня сложности)

В основе игры:
* Python 3
* PyQt5
* Прямые руки
